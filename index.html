Privacy Policy for SpectraBot

Effective Date: 11/11/2025

Contact: hello@spectrabot.site

A. Information We Collect

Data Type Purpose of Collection Retention Period
Discord User ID Identifying premium users, caching user experience/level data, and tracking blacklisted users. Retained until user leaves all shared Discord servers or requests deletion.
Discord Guild (Server) ID Storing server-specific configurations (e.g., Automod settings, premium status, license keys, log channel IDs). Retained until the bot is removed from the server.
License Keys/Premium Status Verifying access to premium features and setting expiry dates. Retained for the duration of the license.
Usage Data Tracking command usage statistics for service improvement and troubleshooting. Retained for up to 90 days.
Automod Keywords Storing custom words provided by server administrators for the bot's Automod features. Retained until removed by a server administrator or the bot is removed from the server.

B. Third-Party Services

SpectraBot uses the following third-party services, which may store or process the data mentioned above:

Google Firebase: Used for persistent storage of configuration and user data. Their Privacy Policy governs their practices.

Better Uptime: Used for monitoring bot status and uptime. Data sent is non-identifying (e.g., connection metrics).

Discord API: All data is collected via the Discord API and is subject to the Discord Privacy Policy.

C. Data Security

We implement commercially acceptable methods to protect your data, including storing sensitive data like User IDs in encrypted databases (Google Firebase) and limiting access only to essential bot operations. However, no method of transmission over the internet is 100% secure.

D. Your Rights & Data Deletion

You have the right to request access to and deletion of any personally identifiable data the bot holds on you.

Server Data: Server administrators can remove all stored Guild Data by simply removing the bot from the Discord server.

User Data: To request the permanent deletion of your User ID data, please contact us at data@spectrabot.site. We will confirm and process your request within 30 days.
